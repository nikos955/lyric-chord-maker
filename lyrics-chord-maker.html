📖 How to use:

1. Paste lyrics → click 📥 Load Lyrics 2. Tap any word → pick a chord from the list 3. Use 🎵 Transpose to change key up/down 4. Click 📊 Display to switch between chord names and intervals 5. Select the Key for interval display (e.g., if song is in G, select G) 6. Click 💾 Save Chords → saves as .json 7. Click 📂 Load Chords → select .json to restore 8. Click 📤 Export .txt → creates .txt for sharing
💾 Save Chords = saves as .json
📂 Load Chords = restores from .json
📤 Export .txt = saves as .txt
